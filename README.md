# P3.Jonathan
Práctica 3 de Jonathan para DevOps: Proyecto que incluye uso básico de PHP, conexión con BBDD MySQL con Página HTML y CSS básico cargados mediante PHP.

# AUTOR: Jonathan Menocal

El creador de este Proyecto es el Alumno Jonathan Menocal, de DAW 2.

## Requisitos iniciales

Se ha utilizado PHP 5, MySQL 5, HTML 5 y CSS 3.

Se requiere contar con una BBDD (Base de Datos) a la que conectar, que debe contar con un Usuario y Contraseña.
Puede ser MySQL, MariaDB, MongoDB, etc.

Se requiere el uso de Servidor Web compatible con PHP para correr el Proyecto.
